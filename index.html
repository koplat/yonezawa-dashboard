02 — Real Data · GW
置賜GW人流レポート 2026
本タブは置賜GW人流レポート2026の実数値を使用したモックアップ基準ページ。出典：agoop（株式会社アグープ）／GW期間（4/28〜5/5、各8日間）／5/2〜5/5は速報値
Executive Summary · 要約

山形県内 ▲5.1%、米沢市内 ▲14%。ただし「県外流出」という単純構図ではなく、米沢市内の弱含みと周辺の好調が同時進行している。

📊 主要発見（What）
  • 米沢市内 ▲14% が県全体を押し下げ：上杉神社▲20%、白布温泉▲30%、川中島合戦会場▲50%、米沢駅▲14%、小野川温泉▲11%
  • 置賜内（米沢市外） +23% が大健闘：熊野大社（南陽）+85% が圧倒的牽引、白川湖▲13%
  • 山形県内（置賜外） +5% で安定：天童温泉+6%、ダイワロイネットホテル山形+9%、上山▲12%
  • 参考：道の駅ふくしま▲2%（前年並み）。県外への流出という単純構図ではない
💡 示唆・申し送り（So What）
  • 川中島合戦会場の▲50% は2025年5/3のイベントスパイク不在による集計差で、実態的な集客力低下ではない。5/3の上杉まつり実施有無の影響を可視化する分析が要
  • 白布温泉▲30% は冷涼な高地で天候影響を受けやすい立地。一方、ダイワロイネットホテル山形（山形市）+9%との対比は都市型宿泊の堅調を示す
  • 熊野大社+85% は「米沢→南陽」の周遊形成のポテンシャル。米沢発の周遊商品設計に活かせる
  • 時間帯別では米沢駅は休日12時ピーク・分散来訪、道の駅米沢はランチ集中型。立地特性に応じたコンテンツ設計が有効

Executive Summary · 要約

12エリアで延べ来訪 4,229人、米沢市内56%・置賜内合算65%。県外比率14%（うち福島・南東北が大半）。県外送客より「近隣集客の質を上げる」段階。

📊 主要発見（What）
  • エリア別TOP：花角味噌醸造 763人（18.0%）、平山孫兵衛538、Nitorito 535、紅花染新田508、米沢織物おりじん473、粟野商事JUYOKAN 374
  • 居住地：米沢市民が中心（2,360人・55.8%）、福島市155、南陽市143、只見町138、小国町134、高畠町86
  • 県外比率：約14%（福島県345＋栃木県33＋茨城県31＋埼玉県25＋東京都12＋他）— 知名度の伸びしろあり
  • 平日と休日で属性が大きく異なる：平日30-60代社会人、休日10代以下＋60代家族
💡 示唆・申し送り（So What）
  • 山形大学工学部は通常授業日の人流（学生・教職員）が含まれるため除外。本数値はOF会場（参加企業）のみの集計
  • 県外14%は伸びしろ。福島県（345人）は地理的近さの効果、北関東（栃木33・茨城31）の存在は地味だが重要。東京都12人は首都圏PRがこれから
  • 休日（9/13土）が最大集客（2,171人・51.3%）。土曜の家族層集客が中核、平日のビジネス・関係者導線とは明確に異なる
  • 滞在時間：休日は2-5時間が最多（観光来訪）、平日5-12時間は関係者・学生想定

Claude Code で SNS 反響を自動更新
手動収集ゼロ ／ Yahoo!リアルタイム検索からClaudeが取得・解析・コミットまで実行
イベント終了後、ターミナルで Claude Code を起動して下記のスラッシュコマンドを叩くだけでSNS反響データが最新化されます。Claudeが Yahoo!リアルタイム検索からX投稿を取得し、感情分析・ワードクラウド生成・JSON更新・git pushまで自動実行します。
/sns-update {event-id}
1. Yahoo!リアルタイム検索 → 2. Claude が投稿を抽出 → 3. CSV保存 → 4. 解析（感情・キーワード） → 5. JSON更新 & git push → 6. Netlify自動デプロイ

セットアップ手順（初回のみ）
Claude Code を最初に使うときのセットアップ。一度設定すれば以降はワンコマンドで運用可能
  1. Claude Code をインストール（macOS / Linux / Windows対応）
  2. 本リポジトリをクローン：git clone <repo-url> && cd yonezawa-dashboard
  3. 依存パッケージをインストール：npm install
  4. ターミナルで claude と入力して Claude Code を起動
  5. スラッシュコマンドを実行（上記の COPY ボタンから貼付）
補足： このスラッシュコマンドの実体は .claude/commands/sns-update.md に定義されています。Claude Code はこのファイルを読み、書かれている手順を順番に実行します。手順を変更したい場合はこのMarkdownを編集してください。
